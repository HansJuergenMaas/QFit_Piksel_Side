## How to write a User Story

When a feature moves from Idea to Ready:

```
User Story: [ID] - [Title]

As a [who] I want to [do what],
so that [benefit / why].

Acceptance Criteria:
- Criterion 1
- Criterion 2

Design Notes:
- Relevant rules from DESIGN.md

WordPress Notes:
- Considerations for WordPress handoff

Dependencies:
- Other User Stories that must be completed first
```
